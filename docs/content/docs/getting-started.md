# 开始使用

MoriUI 提供一组为 Vue 3 准备的界面组件。组件负责可靠的键盘、焦点与状态交互，Nova 主题负责统一而克制的视觉表达；业务代码只需要组合它们。

## 适用场景

当团队希望共享交互规则、减少重复样式，并继续保留 Vue 组件的组合自由时，可以从 MoriUI 开始。它既适合新页面，也适合按功能区域逐步替换已有界面。

## 基本组成

每个界面由三个部分组成：从 `moriui` 导入的组件、一次性载入的样式入口，以及由业务状态驱动的 Vue 模板。组件不会替你决定数据结构，也不要求额外的全局状态层。
